## 实现思路

### 核心算法
- 使用**回溯法**解决八皇后问题
- 采用一维数组表示棋盘，`board[i]`表示第i行皇后所在的列索引
- `is_safe`函数检查放置皇后是否安全（无列冲突和对角线冲突）
- `backtrack`函数递归尝试在每一行放置皇后

### 功能特点
- 支持任意大小的N皇后问题（N≥1）
- 返回所有可能的解决方案
- 提供`print_solution`函数以人类可读的格式打印解决方案

## 运行方式

### 直接运行求解器
```bash
python src/n_queens.py
```

这将求解8皇后问题并打印前3个解决方案。

### 运行单元测试
```bash
python -m pytest tests/test_n_queens.py -v
```

测试用例包括：
- N=1（基本情况）
- N=2和N=3（无解决方案的情况）
- N=4（已知2个解决方案）
- N=8（经典8皇后问题，92个解决方案）
- 解决方案有效性验证（确保所有解决方案都是有效的）
